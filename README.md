# data_analysis
Репозиторий с выполненными лабораторными работами по предмету "Интеллектуальный анализ данных" 6 семестр НИТУ МИСИС.
Кратко про каждую лабораторную работу:
## Лабораторная работа №1: Цифровой советчик сталевара
Разработала программу, которая рекомендует сталевару оптимальное количество ферросплавов для легирования стали. Обработала производственные данные о плавках (объединение таблиц, очистка от пропусков и дубликатов), построила гистограммы и матрицу корреляции для анализа данных. Рассчитала экономический эффект — перерасход сплавов составил 23,2 млн руб. за месяц. Обучила 5 моделей CatBoost для предсказания содержания элементов (Cr, Ni, Mo, V, W) в готовой стали. Создала функцию-советчик, которая подбирает количество сплава перебором с использованием модели. Потенциальная годовая экономия — около 278 млн руб.
## Лабораторная работа №2: Обнаружение ям по телеметрии самосвала
По данным с датчиков карьерного самосвала (GPS, ускорения, давление в подвеске) разработан алгоритм автоматического поиска ям на дороге. Сигнал давления очищен от профиля дороги и шума с помощью FFT-фильтрации. В очищенном сигнале найдены резкие отрицательные пики — ямы. Результат визуализирован на интерактивной карте OpenStreetMap.
