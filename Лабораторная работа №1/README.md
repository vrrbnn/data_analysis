# Цифровой советчик сталевара
Была создана программа-советчик, которая помогает сталевару рассчитать, сколько нужно добавить ферросплавов в сталь, чтобы получить нужный химический состав.

## Зачем это нужно
При производстве стали в нее добавляют разные металлы (хром, никель, молибден, ванадий, вольфрам), чтобы придать нужные свойства. Эти металлы очень дорогие. Если добавить слишком много — перерасход денег. Если добавить мало — сталь не соответствует требованиям.

Советчик помогает точно рассчитать нужное количество.

## Что было сделано по шагам
1. Собрала и подготовила данные
Взяла данные о плавках из Excel-файла (несколько таблиц объединила в одну). Удалила дубликаты и пропуски. Удалила столбцы, где все значения одинаковые (дисперсия равна 0), т.к. они не несут полезной информации.

2. Посмотрела, как связаны разные параметры
Построила график корреляции — он показывает, какие параметры влияют друг на друга. Например, видно, что чем больше добавили феррохрома, тем выше содержание хрома в готовой стали. Это логично и значит, что данные правильные.

3. Построила гистограммы
Посмотрела, как часто сталевары перебарщивают с добавками. Оказалось, что по хрому и никелю перерасход встречается часто — гистограммы смещены вправо от нуля.

4. Посчитала экономический эффект
Рассчитала, сколько ферросплавов было добавлено лишних (сверх необходимого).

5. Обучила модели машинного обучения
Для каждого из 5 элементов обучила модель CatBoost, которая предсказывает, каким будет содержание элемента в стали, если добавить определенное количество сплава.

6. Создала советчик
Написала функцию, которая:
1) Смотрит, сколько элемента уже есть в стали (в полупродукте)
2) Смотрит, сколько должно быть по технологии (целевое значение — середина между нижней границей и целевым)
3) Если нужно добавить — подбирает количество сплава перебором (от 0 до 5000 кг с шагом 100 кг)
4) Для каждого варианта спрашивает у модели, что получится
5) Выбирает подходящее количество

## Что получилось в итоге
Советчик выдает рекомендации для каждой плавки. Пример:
Плавка: добавить 3700 кг феррохрома (текущее содержание 11.250%, нужно 11.580%)
Проверила на 5 случайных плавках — советчик работает, дает конкретные цифры.

## Главный вывод
Внедрение такого советчика на заводе позволит:
1) Точнее попадать в нужный состав стали
2) Сэкономить ~278 млн рублей в год на ферросплавах
3) Уменьшить количество брака
